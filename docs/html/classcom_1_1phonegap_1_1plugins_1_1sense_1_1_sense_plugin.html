Detailed Description

PhoneGap plugin implementation for the Sense Platform. Provides PhoneGap applications with an interface to the native Sense service from JavaScript.

Author
Steven Mulder steve.nosp@m.n@se.nosp@m.nse-o.nosp@m.s.nl
