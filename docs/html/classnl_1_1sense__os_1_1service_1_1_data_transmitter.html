Detailed Description

This class is responsible for initiating the transmission of buffered data. It works by setting periodic alarm broadcasts that are received by this class. The Sense service calls scheduleTransmissions(Context) when it starts sensing.

When the transmission alarm is received, an Intent is sent to the MsgHandler to empty its buffer.

The transmission frequency is based on the Main#SYNC_RATE preference. When the sync rate is set to the real-time setting, we look at the and Main#SAMPLE_RATE to determine periodic "just in case" transmissions.
