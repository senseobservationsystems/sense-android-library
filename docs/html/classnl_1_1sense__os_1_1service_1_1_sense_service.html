Detailed Description

Main Sense service class.

Activities can bind to this service and call functions to:

  • log in;
  • register;
  • start sensing;
  • start/stop individual sensor modules;
  • set and get properties;

When the toggleMain(boolean) method is called to start the sensing, the service starts itself and registers itself as a foreground service so it does not get easily killed by Android.

Starts the Sense service. Tries to log in and start sensing; starts listening for network connectivity broadcasts.

Parameters
intentThe Intent supplied to Activity#startService(Intent). This may be null if the service is being restarted after its process has gone away.
