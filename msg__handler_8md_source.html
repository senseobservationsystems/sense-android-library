1 # Message Handler
2 
3 MsgHandler is a service that responsible for handling the data that has been collected by different sensors. There are two task that it could done :
4 * Receive sensor data from sensors and store it in [LocalStorage](docs/storage.md)
5 * Periodically transmit all sensor data in LocalStorage to CommonSense
6 
7 MsgHandler is implemented in nl.sense_os.service.MsgHandler.
8 
9 ## Store datapoint from Sensor
10 
11 To send a sensor data to MsgHandler, sensors need to send an Intent with *action_sense_new_data* that contain the details of the datapoint (see Sensor DataPoint)
12 
13 Here is an example of sending a sensor datapoint to Message Handler.
14 
15  Intent sensorData = new Intent(context.getString(R.string.action_sense_new_data));
16  sensorData.putExtra(DataPoint.SENSOR_NAME, SensorNames.NOISE);
17  sensorData.putExtra(DataPoint.VALUE, BigDecimal.valueOf(dB).setScale(2, 0).floatValue());
18  sensorData.putExtra(DataPoint.DATA_TYPE, SenseDataTypes.FLOAT);
19  sensorData.putExtra(DataPoint.TIMESTAMP, startTimestamp);
20  sensorData.setPackage(context.getPackageName());
21  context.startService(sensorData);

24 ## Transmit sensor data to CommonSense
25 
26 Sense Service using nl.sense_os.service.ctrl.Controller instance to schedule sending of data in LocalStorage to CommonSense periodically. This Controller will create an instance of nl.sense_os.service.DataTransmitter that will registering itself to scheduler to run at particular interval based on nl.sense_os.service.constants.SensePrefs.Main.SyncRate settings in preferences. This DataTransmitter then will send an *action_sense_send_data* intent to nl.sense_os.service.MsgHandler when scheduled to run, to actually send the sensor data to CommonSense and empty itâ€™s buffer.
27 
28 This is available option for SyncRate settings:
29 * nl.sense_os.service.SensePrefs.Main.SyncRate.RARELY (15 minute)
30 * nl.sense_os.service.SensePrefs.Main.SyncRate.ECO_MODE (30 minute)
31 * nl.sense_os.service.SensePrefs.Main.SyncRate.NORMAL (5 minute)
32 * nl.sense_os.service.SensePrefs.Main.SyncRate.OFTEN (1 minute)
33 * nl.sense_os.service.SensePrefs.Main.SyncRate.REAL_TIME (depend on sample rate)
